# SmrtLocal-Assistant
使用 LM-Studio 的本地 API 制作的桌面 AI 小助手
一个基于 PyQt5 和 LMStudio 构建的本地 AI 助手桌面应用，支持与本地部署的大语言模型进行交互，无需联网即可使用。
功能特点
本地运行：通过 LMStudio 与本地大语言模型交互，保护隐私
简洁界面：直观的聊天界面，支持深色 / 浅色模式切换
模型管理：可选择、保存和管理多个本地模型
格式化显示：支持 Markdown 格式的消息展示，包括代码块、标题、列表等
对话管理：支持清空对话、导出对话记录、复制内容等功能
自定义设置：可调整字体大小和类型，适应不同使用习惯
前提条件
Python 3.6 或更高版本
已安装并运行 LMStudio（默认 API 地址：http://127.0.0.1:1234）
已在 LMStudio 中加载至少一个语言模型
安装步骤
克隆或下载本项目代码
安装所需依赖：
bash
pip install PyQt5 requests
使用方法
确保 LMStudio 已启动并加载了模型
运行应用程序：
bash
python lm_desktop_assistant.py
在模型下拉框中选择已加载的模型
在输入框中输入问题，按 Enter 发送（Shift+Enter 可换行）
可通过工具栏和菜单使用各种功能
主要功能说明
模型管理：
从下拉框选择已加载的模型
点击 "保存模型" 将当前模型添加到列表中
对话操作：
"清空对话"：清除当前所有对话历史
"复制全部"：复制所有对话内容到剪贴板
右键点击对话区域可复制单条消息
视图设置：
支持浅色 / 深色模式切换
可自定义字体和字号
导出对话记录为文本文件
界面说明
左侧：对话显示区域，展示用户与 AI 的助手的交流内容
右侧：信息面板，显示当前模型、历史消息计数和响应进度
底部：输入区域，用于输入问题和调整字体大小
注意事项
确保 LMStudio 服务正在运行且 API 地址正确（默认：http://127.0.0.1:1234）
首次使用时，应用会尝试从 LMStudio API 获取已加载的模型列表
如遇到连接问题，请检查 LMStudio 是否正常运行及网络设置
